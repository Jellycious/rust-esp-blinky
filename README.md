# ESP32 Blinky in Rust

## Development Environment
A Dockerfile and a few utility scripts are provided to setup a docker development environment.

Build the container with: `docker build -t <container_name>`. 

Make sure to replace `<container_name>` with your desired container name.

Update `CONTAINER_NAME=<container_name>` variable in `dev-env.sh` and `runner.sh` script.
